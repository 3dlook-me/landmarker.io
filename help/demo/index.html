landmarker.io is designed to be used with a command line asset server called landmarkerio-server. This is what actually loads up meshes and landmarks from your computer and what saves the landmarks back to disk.

Upon starting, landmarker.io looks for the server running on your machine - in particular, at localhost:5000. If can't find it, it restarts in a demo mode, to show the functionality of the landmarker.

Having trouble using landmarkerio-server? Check out the README on it's github page.
