t toggle texture (if there is texture)
q clear all selections
d Delete selected landmarks
a Select all landmarks in group
space Reset camera to default
c Change between orthographic and perspective rendering (mesh only)
j Move to the next asset
k Move to the previous asset
click on landmark Select a landmark
click + drag on landmark Move landmark points
shift + drag NOT on landmark Draw a box to select multiple.
ctrl/cmd + click on landmark Select and deselect from current selection.
